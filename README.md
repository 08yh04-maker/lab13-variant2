## Название работы
Мультиагентные системы: разработка распределённых интеллектуальных агентов

## Описание проекта
Мультиагентная система для автоматизации техподдержки. Агенты:
- **Классификация тикетов** — определяет категорию и приоритет
- **Поиск по базе знаний** — находит похожие решения
- **Генерация ответов** — формирует ответ пользователю
- **Эскалация** — передаёт тикет человеку при сложности

## Технологии
- **Go** (агенты) — nats.go
- **Python** (оркестратор) — nats-py, asyncio, FastAPI
- **NATS** — брокер сообщений
- **Redis** — состояние агентов
- **Jaeger** — распределённая трассировка
- **Docker, Docker Compose**
